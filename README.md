# Flowin-App
Flowin 📋 *Ainda em desenvovimento*
O Flowin é um ecossistema integrado de produtividade desenvolvido em Python utilizando o framework Streamlit. O objetivo central da aplicação é consolidar diversas ferramentas de organização pessoal — foco, finanças e gestão de tarefas — em uma interface única, moderna e intuitiva.

🚀 Funcionalidades Principais
🙂 Verity IA: Assistente virtual integrada baseada na API do Gemini, capaz de analisar seu contexto de dados, oferecer insights sobre finanças e ajudar na organização de rotinas.

✅ Gerenciador de Tarefas: Sistema de criação, priorização e acompanhamento de compromissos diários.

💰 Controle Financeiro: Monitoramento de despesas por categoria com visualização dinâmica via gráficos Plotly.

📓 Bloco de Notas: Espaço para registro de ideias, rascunhos e logs de desenvolvimento, garantindo que nenhum insight se perca.

⏱️ Cronômetro Pomodoro: Ferramenta de foco baseada em ciclos de trabalho de 25 minutos, auxiliando na gestão de tempo e produtividade.

🛠️ Tecnologias Utilizadas
Linguagem: Python

Framework de Interface: Streamlit

Inteligência Artificial: Google Gemini API (modelo gemini-2.0-flash)

Visualização de Dados: Plotly

Gestão de Estado: Session State do Streamlit para persistência de dados durante a sessão.
